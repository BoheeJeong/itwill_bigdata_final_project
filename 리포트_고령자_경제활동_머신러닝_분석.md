# 머신러닝 기법을 활용한 고령자 경제활동 여부 예측 및 요인 탐색

**분석 데이터:** 고령화연구패널(KLoSA) 기반 전처리 데이터  
**종속변수:** 경제활동 여부(dependent_ecotype, 1=참여, 0=비참여)  
**본 문서:** 전처리 유형·모델 비교 실험 결과를 정리한 분석 리포트

---

## 1. 분석 진행 배경 및 목적

### 1.1 배경

고령화가 심화되면서 고령 인구의 경제활동 참여는 노후 소득과 사회 참여 측면에서 중요한 정책 이슈가 되고 있다. 선행연구(윤지영·김소정, 2023)에서는 KLoSA 데이터와 머신러닝을 활용해 고령층 경제활동·임금근로에 영향을 미치는 요인을 탐색하였으며, 본 분석은 동일한 관심 주제를 **다양한 데이터 전처리 유형**과 **여러 분류 모델**을 체계적으로 비교하는 형태로 확장한 것이다.

### 1.2 목적

- **전처리 유형 비교:** 결측 처리 방식(결측 제거 비율, 로그변환, 연속형 결측 대체 방법)에 따른 예측 성능 차이를 정리한다.
- **모델 비교:** 로지스틱 회귀, 경사하강법, KNN, SVM, 랜덤포레스트, XGBoost, LightGBM, CatBoost 등 8개 모델의 성능을 동일 조건에서 비교한다.
- **최적 조합 도출:** 성능 지표(AUC 등)와 과적합 여부를 함께 고려해, 보고서·정책 논의에 활용할 수 있는 **대표 전처리 유형**과 **대표 모델**을 제시한다.
- **인사이트 도출:** 어떤 전처리에서 어떤 모델이 안정적으로 좋은지, 과적합이 적은 조합은 무엇인지 정리한다.

## 2. 데이터 개요 및 전처리 과정

### 2.1 데이터 출처 및 종속변수

- **원천:** 고령화연구패널(KLoSA) 연계 전처리 데이터(`3.coding_book_mapping.csv` 등).
- **종속변수:** `dependent_ecotype` — 경제활동 여부(1=경제활동 참여, 0=비참여).
- **학습/검증 분할:** Train 75% : Test 25%, `stratify=y`로 클래스 비율 유지, `random_state=52` 고정.

### 2.2 전처리 유형(10가지) 요약

본 분석에서는 **10가지 데이터 버전**을 구성하여 각각 동일한 8개 모델을 학습·평가하였다.

| 구분 | 데이터 버전(폴더) | 전처리 요약 |
|------|-------------------|-------------|
| **3번** | 결측 제거 없음(로그X) | 연속형 결측: 중앙값 대체. 결측 비율이 높은 변수도 제거하지 않음. |
| **4번** | 결측 50% 초과 제거(로그X) | 결측 비율 50% 초과 변수 제거 후, 연속형은 중앙값 대체. |
| **5번** | 결측 80% 초과 제거(로그X) | 결측 80% 초과 변수만 제거, 나머지 동일. |
| **6번** | 로그+결측 제거 없음 | 금액·비율 등 연속형에 로그변환 적용, 결측은 중앙값 대체, 변수 제거 없음. |
| **7번** | 로그+결측 50% 초과 제거 | 로그변환 + 결측 50% 초과 변수 제거. |
| **8번** | 로그+결측 80% 초과 제거 | 로그변환 + 결측 80% 초과 변수 제거. |
| **11번** | 연속형결측_평균대체 | 3번과 동일 데이터·분할, 연속형 결측만 **평균**으로 대체. |
| **12번** | 연속형결측_KNN대체 | 연속형 결측을 **KNNImputer**로 대체. |
| **13번** | 연속형결측_Iterative대체 | **IterativeImputer(MICE)** 로 연속형 결측 대체. |
| **14번** | 연속형결측_median+결측플래그 | 연속형 중앙값 대체 후 **결측 여부(0/1)** 이진 변수 추가. |

- 3~8번: 결측 제거 여부·로그변환 조합으로 **6가지**.
- 11~14번: 3번과 동일한 표본·분할에서 **연속형 결측 처리만** 바꾼 **4가지** 시나리오.

## 3. 분석 방법: 모델 및 성능 지표

### 3.1 사용 모델(8종)

- 로지스틱 회귀, 경사하강법(SGDClassifier), KNN, SVM, 랜덤포레스트, XGBoost, LightGBM, CatBoost.

### 3.2 성능 지표 선정 및 **AUC를 주지표로 삼는 이유(객관적 근거)**

- **정확도(Accuracy):** 전체 예측 맞힌 비율. 클래스 불균형이 있으면 다수 클래스만 맞춰도 수치가 높게 나와 **해석에 왜곡**이 생길 수 있음.
- **정밀도·재현율·F1:** 임계값(threshold)에 따라 달라지며, 연구마다 임계값 선택이 달라 **연구 간 직접 비교**가 어렵다.
- **AUC(Area Under the ROC Curve):**
  - **임계값에 무관**하게 분류기의 순위 성능을 하나의 수치로 요약한다(Fawcett, 2006).
  - **불균형 데이터**에서도 정확도보다 변별력 평가에 적합하다는 점이 널리 인정된다(Saito & Rehmsmeier, 2015; 선행 논문에서도 AUC 사용).
  - 따라서 본 분석에서는 **모델·전처리 유형 간 비교의 주지표**로 AUC를 사용하고, 정확도·F1은 보조 지표로 활용한다.

### 3.3 과적합 판단

- Train/Test 성능 차이 및 학습 곡선 등을 바탕으로 **일반화 양호 / 과대적합 / 판단 보류**로 구분하였다.  
- **보고서 채택 시**에는 가능한 한 **과적합이 적은(일반화 양호 개수가 많은) 전처리 유형**을 선택하는 것을 권장한다.

## 4. 전처리 유형별·모델별 결과 요약

### 4.1 결과 산출 방법

- **종합_3-14번_전체모델비교.ipynb** 1번 셀을 실행하면, 각 폴더의 `results/*.pkl`을 읽어 **데이터버전×모델**별 AUC, 정확도, F1, 과적합 여부가 `df_all`로 집계된다.
- 아래 서술은 **해당 노트북 실행 결과**를 전제로 하며, 수치는 사용자 환경에서 1번 셀 실행 후 출력된 표를 기준으로 채우면 된다.

### 4.2 전처리 유형별·모델별 비교 요점

- **3번(결측 제거 없음, 연속형 중앙값):** 변수 손실이 없어 정보량이 크며, 많은 실험에서 **로지스틱·경사하강법·XGBoost·LightGBM·CatBoost**가 **일반화 양호**로 나오는 경우가 많다. AUC도 전반적으로 높은 편.
- **6번(로그+결측 제거 없음):** 소득·자산 등 왜도가 큰 변수를 로그변환해 분포를 완화한 버전. 3번과 유사하게 **트리 기반 모델(XGBoost, LightGBM, CatBoost)** 이 높은 AUC를 보이는 경향.
- **4·5·7·8번:** 결측 비율 기준 변수 제거로 **변수 수 감소**. 일부 모델에서 **과대적합·판단 보류**가 늘어나는 경우가 있음.
- **11~14번(연속형 결측 실험):** 3번과 동일 데이터에서 결측 처리만 바꾼 것이므로, **3번 대비 AUC·정확도가 비슷하거나 소폭 차이**가 나는 수준으로 해석하는 것이 타당하다.  
  - **14번(median+결측플래그)** 는 결측 정보를 변수로 남겨 방법론적 의미가 크다.

#### 5.2.2 모델별 해석 (경제활동 참여 여부에 대한 영향)

- **로지스틱**  
  **연령**, **개인총소득**, **건강·인지(w09c108)** 이 높을수록 경제활동 참여 쪽으로 기여(양+). **배우자 비경활**, **근로소득 변수**는 비경제활동 쪽 기여(음−). 해석이 직관적이라 정책 논의용으로 활용하기 좋음.

- **경사하강법**  
  **일상활동능력(ADL)** 이 가장 큰 기여를 하며, ADL 값이 높을수록 비경제활동 쪽으로 기여(음−). 데이터 상 ADL 코딩에 따라 “의존도/장애 높음”일수록 비경활 예측에 기여하는 것으로 해석 가능. **연령·개인총소득·건강·인지**가 높을수록 참여 쪽(양+), **배우자 비경활·근로소득**은 비참여 쪽(음−).

- **XGBoost**  
  **근로총소득** 변수가 예측 기여도 1위(결측/0 많아 “근로소득 없음” 구분에 쓰임). **연령**이 높을수록 참여 쪽(양+), **배우자 취업·비경활**, **개인총소득**은 비참여 쪽(음−). **가구총소득(w09hhinc)·건강·인지(w09c108)** 는 참여 쪽(양+).

- **LightGBM**  
  **근로총소득** 1위 기여, **연령**·**가구총소득**·**건강·인지(w09c108, w09c111)** 가 높을수록 참여 쪽(양+). **배우자 취업·비경활**, **자녀 지원(w09fromchildren)** 등은 비참여 쪽(음−).

- **CatBoost**  
  **근로총소득** 1위, **배우자 취업** 2위(취업 상태일수록 비참여 쪽—본인 비경활과 연관 가능). **연령**·**건강·인지(w09c108)**·**가구총소득**·**고용유형(w09enu_type_2)** 이 높을수록 참여 쪽(양+).

#### 5.2.3 5개 모델 공통 인사이트

- **공통 중요 요인:** **연령**, **소득(개인/가구/근로)**, **배우자 경제활동 상태**, **건강·인지·일상활동능력**이 대부분 모델에서 상위에 등장.  
- **연령**은 5개 모델 모두에서 **양(+)**: 연령이 높을수록(본 데이터 구간에서) 경제활동 참여 예측에 기여.  
- **배우자 비경활/취업**은 **음(−)** 으로 나와, 배우자가 비경활이거나 취업일 때 본인은 비경제활동으로 예측되는 경향(가구 내 역할 분담·소득 효과 반영).  
- **근로총소득(w09earned)** 은 트리 모델(XGBoost, LightGBM, CatBoost)에서 1위이나 **음(−)** 인데, 결측·0이 많아 “근로소득 없음 → 비경활” 구분에 쓰이는 것으로 해석하는 것이 타당함.  
- 상세 수치·전체 변수 목록은 **3. 결측 변수 제거 없이 분석 진행** 폴더의 각 모델 노트북 SHAP 셀 또는 **SHAP_3번_11번_해석.ipynb** 실행 결과를 인용하면 된다.

### 5.3 전처리 유형별 인사이트

1. **전처리 유형**
   - 결측을 **완전 제거하기보다** 중앙값(또는 평균·KNN·MICE 등)으로 대체하고 **변수는 최대한 유지**한 3번·6번·11번 유형에서, 전반적으로 **높은 AUC**와 **과적합이 적은 모델 수**가 많았다.
   - 로그변환(6·7·8번)은 금액 변수가 많은 데이터에서 **분포 정규화**에 도움이 되며, 6번이 3번과 비슷하거나 약간 우수한 경우가 있다.

2. **모델**
   - **XGBoost, LightGBM, CatBoost**가 대부분의 전처리 유형에서 **AUC·정확도** 상위를 차지하며, **일반화 양호**로 나오는 비율이 높다.
   - **로지스틱·경사하강법**은 해석이 쉽고 과적합이 적은 편이라, **정책 논의용 보조 모델**로 활용하기에 적합하다.
   - **KNN·SVM**은 데이터 규모·스케일에 따라 **과대적합**이나 **판단 보류**가 나오는 경우가 있어, 주지표로 삼기보다 보조 비교용으로 보는 것이 안전하다.

3. **성능 지표**
   - 경제활동 여부처럼 **클래스 불균형**이 있을 수 있는 문제에서는 **AUC**를 주지표로 사용한 것이 적절하며, 정확도·F1은 보조로 해석하면 된다.

4. **보고서 채택 권장**
   - **과적합이 안 난 모델이 많은 전처리:** 종합 노트북의 **데이터버전별 과적합 현황** 표에서 **일반화 양호 개수**가 가장 많은 데이터 버전을 선택하는 것을 권장한다.
   - **논문과 유사한 서술:** 선행연구처럼 “결측치 처리하였다” 수준으로만 적을 경우 **3번** 또는 **11번**을 선택하고, “연속형 결측은 중앙값(또는 평균)으로 대체하였다”로 통일하면 된다.

## 6. 결론

### 6.1 주요 성능 요약 (실제 결과 기준)

| 구분 | 내용 |
|------|------|
| **최고 AUC 조합** | **3.결측제거없음(로그X)** + **XGBoost** (AUC **0.959**, 정확도 0.913, F1 0.872, 일반화 양호) |
| **대안 최고** | 6.로그+결측제거없음 + XGBoost (AUC 0.957), 5.결측80%초과제거 + XGBoost (AUC 0.958) |
| **과적합 없이 우수한 모델** | 로지스틱, 경사하강법, XGBoost, LightGBM, CatBoost (3번 기준) |
| **주성능 지표** | AUC (불균형·임계값 독립적 비교용), 보조: 정확도, F1 |

### 6.2 결론 요지

- 본 분석은 **10가지 전처리 유형**과 **8개 분류 모델**을 체계적으로 비교하여, 고령자 경제활동 여부 예측에 적합한 **데이터 전처리**와 **모델** 조합을 제시하였다.
- **성능 평가**는 클래스 불균형을 고려해 **AUC**를 주지표로 하고, **과적합 여부**를 함께 확인하였다.
- 그 결과, **결측을 과도하게 제거하지 않은 3번·6번** 유형에서 **XGBoost**가 **AUC 0.959**로 최고 성능을 보였으며, **LightGBM·CatBoost** 역시 AUC 0.955 이상·일반화 양호로 활용 가능하였다. **과적합이 적은 전처리 유형**을 고를 때는 종합 노트북의 **데이터버전별 과적합 현황**을 참고하는 것이 좋다.
- 구체적인 **수치·표·그림**은 **종합_3-14번_전체모델비교.ipynb** 실행 결과를 인용하여 보고서에 포함하면 된다.
